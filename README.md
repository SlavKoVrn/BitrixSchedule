#Тестовое задание

##Контекст

У нас есть продукт «Персональный сайт врача», разработанный на базе 1С-Битрикс.
В продукте уже присутствует базовый модуль записи на приём к врачу, однако сейчас он реализован в упрощённом виде — фактически на уровне вёрстки 
и без административного интерфейса для управления расписанием.

Посмотреть решение можно здесь:
[https://marketplace.1c-bitrix.ru/solutions/vit.doctor/#tab-comments-link](https://marketplace.1c-bitrix.ru/solutions/vit.doctor/#tab-comments-link)

Решение можно установить бесплатно и использовать для выполнения тестового задания.

##Задача

####Необходимо реализовать управление расписанием врача.

Нужно создать интерфейс в Bitrix, в котором можно настроить рабочую неделю врача:
— задать рабочие дни недели
— указать время начала и окончания работы
Пример:
Понедельник — 09:00–18:00
Вторник — 09:00–18:00
Среда — выходной

####Важно

Текущую реализацию модуля можно не использовать.
Допускается предложить и реализовать собственную архитектуру хранения и управления расписанием.
Важно кратко обьяснить выбранный подход в решении.
Ограничение по времени
Ориентир на выполнение задания — до 3 часов.
Если решение не завершено полностью, можно прислать частично готовый результат.
